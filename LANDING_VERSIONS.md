## File Naming Convention
```

- course-landing-v1.0.html
- course-landing-v1.1.html (minor update)

```

## Notes
- Always keep the **unversioned file** as the "latest" (e.g., `course-landing.html`)
- Create a **versioned copy** before making major changes
- Update this tracker when creating new versions
